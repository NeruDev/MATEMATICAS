# Aplicaciones de la Derivada

## Descripción

Este tema cubre las aplicaciones prácticas del cálculo diferencial: optimización, razones relacionadas, análisis de gráficas, aproximaciones lineales y el análisis completo de funciones.

## Subtemas

| Código | Tema | Descripción |
|--------|------|-------------|
| 3.1 | Recta tangente y normal | Ecuaciones de rectas tangentes y normales |
| 3.2 | Razones relacionadas | Cantidades que cambian con el tiempo |
| 3.3 | Valores extremos | Máximos y mínimos absolutos y relativos |
| 3.4 | Criterio primera derivada | Crecimiento, decrecimiento, extremos |
| 3.5 | Criterio segunda derivada | Concavidad y puntos de inflexión |
| 3.6 | Optimización | Maximizar/minimizar con restricciones |
| 3.7 | Aproximaciones y diferenciales | Aproximación lineal, errores |
| 3.8 | Análisis completo | Dominio, asíntotas, extremos, gráfica |
| 3.9 | Newton-Raphson | Aproximación de raíces |
