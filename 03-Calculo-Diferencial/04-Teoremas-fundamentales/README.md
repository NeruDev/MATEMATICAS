# Teoremas Fundamentales del Cálculo Diferencial

## Descripción

Este tema cubre los teoremas que fundamentan el cálculo diferencial: Rolle, valor medio, L'Hôpital para formas indeterminadas, y las series de Taylor/Maclaurin para aproximación de funciones.

## Subtemas

| Código | Tema | Descripción |
|--------|------|-------------|
| 4.1 | Teorema de Rolle | Si f(a)=f(b), existe c con f'(c)=0 |
| 4.2 | Teorema del valor medio | Pendiente instantánea = pendiente secante |
| 4.3 | Valor medio generalizado | Extensión de Cauchy |
| 4.4 | Regla de L'Hôpital | Formas 0/0 y ∞/∞ |
| 4.5 | Otras formas indeterminadas | 0·∞, ∞-∞, 0⁰, 1^∞, ∞⁰ |
| 4.6 | Teorema de Taylor | Aproximación polinomial |
| 4.7 | Series de Maclaurin | Expansión alrededor de x=0 |
