# La Derivada

## Descripción

Este tema cubre el concepto fundamental de la derivada como límite del cociente incremental, su interpretación geométrica como pendiente de la recta tangente, y las técnicas de diferenciación para todo tipo de funciones.

## Subtemas

| Código | Tema | Descripción |
|--------|------|-------------|
| 2.1 | Definición de derivada | Límite del cociente incremental |
| 2.2 | Derivadas elementales | Potencias, exponenciales, logarítmicas, trigonométricas |
| 2.3 | Reglas de diferenciación | Suma, resta, producto, cociente |
| 2.4 | Regla de la cadena | Derivada de funciones compuestas |
| 2.5 | Derivación implícita | Ecuaciones que definen funciones implícitamente |
| 2.6 | Derivadas de orden superior | Segunda derivada y derivadas sucesivas |
| 2.7 | Derivadas de funciones inversas | Inversas y trigonométricas inversas |
| 2.8 | Derivación logarítmica | Técnica para productos y potencias complicadas |
