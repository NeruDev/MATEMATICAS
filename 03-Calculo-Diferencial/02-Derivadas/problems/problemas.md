# Problemas de Derivadas

> **Instrucciones:** Calcula la derivada. Simplifica cuando sea posible.

## 2.1 Definición de Derivada

### Problema 2.1.1 ⭐
Usa la definición para calcular $f'(x)$ si $f(x) = 3x - 2$.

### Problema 2.1.2 ⭐
Usa la definición para calcular $f'(x)$ si $f(x) = x^2 + x$.

### Problema 2.1.3 ⭐⭐
Usa la definición para encontrar $f'(4)$ si $f(x) = \sqrt{x}$.

### Problema 2.1.4 ⭐⭐
Usa la definición para calcular $f'(x)$ si $f(x) = \frac{1}{x}$.

### Problema 2.1.5 ⭐⭐⭐
Encuentra la ecuación de la recta tangente a $y = x^2$ en $x = 3$ usando la definición de derivada.

## 2.5 Derivación Implícita

### Problema 2.5.1 ⭐⭐
Encuentra $\frac{dy}{dx}$ si $x^2 + y^2 = 25$.

### Problema 2.5.2 ⭐⭐
Encuentra $\frac{dy}{dx}$ si $xy = 10$.

### Problema 2.5.3 ⭐⭐
Encuentra $\frac{dy}{dx}$ si $x^2 + xy + y^2 = 7$.

### Problema 2.5.4 ⭐⭐⭐
Encuentra $\frac{dy}{dx}$ si $\sin(xy) = x + y$.

### Problema 2.5.5 ⭐⭐⭐
Encuentra $\frac{dy}{dx}$ si $e^{xy} = x - y$.

### Problema 2.5.6 ⭐⭐⭐
Para $x^3 + y^3 = 6xy$, encuentra la pendiente de la tangente en $(3, 3)$.

### Problema 2.5.7 ⭐⭐⭐⭐

## 2.6 Derivadas de Orden Superior

### Problema 2.6.2 ⭐

### Problema 2.6.3 ⭐⭐

### Problema 2.6.5 ⭐⭐⭐

### Problema 2.6.6 ⭐⭐⭐
