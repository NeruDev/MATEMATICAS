# Resumen: La Derivada

## Fórmulas Esenciales

### Derivadas Elementales
| Función | Derivada |
|---------|----------|
| $c$ (constante) | $0$ |
| $x^n$ | $nx^{n-1}$ |
| $e^x$ | $e^x$ |
| $a^x$ | $a^x \ln a$ |
| $\ln x$ | $\frac{1}{x}$ |
| $\log_a x$ | $\frac{1}{x \ln a}$ |

### Derivadas Trigonométricas
| Función | Derivada |
|---------|----------|
| $\sin x$ | $\cos x$ |
| $\cos x$ | $-\sin x$ |
| $\tan x$ | $\sec^2 x$ |
| $\cot x$ | $-\csc^2 x$ |
| $\sec x$ | $\sec x \tan x$ |
| $\csc x$ | $-\csc x \cot x$ |

### Reglas de Diferenciación
- **Suma/Resta:** $(f \pm g)' = f' \pm g'$
- **Producto:** $(fg)' = f'g + fg'$
- **Cociente:** $\left(\frac{f}{g}\right)' = \frac{f'g - fg'}{g^2}$
- **Cadena:** $(f \circ g)' = f'(g(x)) \cdot g'(x)$
