## F√≥rmulas Clave

### Soluci√≥n General
$$\mathbf{X}(t) = c_1\mathbf{v}_1 e^{\lambda_1 t} + c_2\mathbf{v}_2 e^{\lambda_2 t}$$

### Eigenvalores Complejos ($\lambda = \alpha \pm \beta i$)
$$\mathbf{X}(t) = e^{\alpha t}\left[C_1(\mathbf{a}\cos\beta t - \mathbf{b}\sin\beta t) + C_2(\mathbf{a}\sin\beta t + \mathbf{b}\cos\beta t)\right]$$

### Eigenvalor Repetido (Defectivo)
$$\mathbf{X}(t) = C_1\mathbf{v}e^{\lambda t} + C_2(\mathbf{v}te^{\lambda t} + \boldsymbol{\eta}e^{\lambda t})$$

### Clasificaci√≥n de Puntos de Equilibrio

| Tipo | Eigenvalores |
|:-----|:-------------|
| Nodo estable | $\lambda_1, \lambda_2 < 0$ reales |
| Nodo inestable | $\lambda_1, \lambda_2 > 0$ reales |
| Silla | $\lambda_1 < 0 < \lambda_2$ reales |
| Espiral estable | $\alpha < 0$ (complejos) |
| Espiral inestable | $\alpha > 0$ (complejos) |
| Centro | $\alpha = 0$ (imaginarios puros) |
