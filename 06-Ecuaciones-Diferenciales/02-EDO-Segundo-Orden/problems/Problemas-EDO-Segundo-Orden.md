# Problemas: EDO de Segundo Orden

---

## 2.1 Ecuaciones Homogéneas con Coeficientes Constantes

### Raíces Reales Distintas

**Problema 1.** Resolver las siguientes ecuaciones:

a) $y'' - 5y' + 6y = 0$

b) $y'' + y' - 6y = 0$

c) $y'' - 4y = 0$

d) $2y'' - 5y' - 3y = 0$

**Problema 2.** Resolver el PVI: $y'' - 3y' + 2y = 0$, $y(0) = 1$, $y'(0) = 0$

### Raíces Repetidas

**Problema 3.** Resolver las siguientes ecuaciones:

a) $y'' - 4y' + 4y = 0$

b) $y'' + 6y' + 9y = 0$

c) $4y'' + 4y' + y = 0$

**Problema 4.** Resolver el PVI: $y'' - 2y' + y = 0$, $y(0) = 2$, $y'(0) = 1$

### Raíces Complejas

**Problema 5.** Resolver las siguientes ecuaciones:

a) $y'' + 4y = 0$

b) $y'' + 2y' + 5y = 0$

c) $y'' - 4y' + 13y = 0$

d) $y'' + 9y = 0$

**Problema 6.** Resolver el PVI: $y'' + 4y' + 8y = 0$, $y(0) = 1$, $y'(0) = 0$

---

## 2.2 Ecuaciones No Homogéneas - Coeficientes Indeterminados

### Sin Modificación

**Problema 7.** Resolver:

a) $y'' - 3y' + 2y = 6$

b) $y'' + y = x^2$

c) $y'' - 4y = e^{3x}$

d) $y'' + 9y = \cos 2x$

**Problema 8.** Resolver:

a) $y'' - y = e^{2x}$

b) $y'' + 4y = \sin x$

c) $y'' - 2y' + y = x^3$

### Con Modificación (Duplicación)

**Problema 9.** Resolver:

a) $y'' - 2y' + y = e^x$

b) $y'' + 4y = \cos 2x$

c) $y'' - y = e^x$

d) $y'' + y = \sin x$

**Problema 10.** Resolver:

a) $y'' - 4y = e^{2x}$

b) $y'' + 4y = 3\sin 2x$

### Términos Combinados

**Problema 11.** Resolver:

a) $y'' - y = e^x + x$

b) $y'' + y = x + \cos x$

c) $y'' - 3y' + 2y = e^x + e^{2x}$

---

## 2.3 Variación de Parámetros

**Problema 12.** Resolver usando variación de parámetros:

a) $y'' + y = \sec x$

b) $y'' + y = \tan x$

c) $y'' + y = \csc x$

**Problema 13.** Resolver:

a) $y'' - y = \frac{1}{e^x + 1}$

b) $y'' + 4y = \sec 2x$

c) $y'' - 2y' + y = \frac{e^x}{x}$

**Problema 14.** Resolver:

a) $y'' + y = \sec^2 x$

b) $y'' - y = e^x \ln x$

---

## 2.4 Ecuación de Cauchy-Euler

**Problema 15.** Resolver para $x > 0$:

a) $x^2y'' - 2xy' - 4y = 0$

b) $x^2y'' + xy' - y = 0$

c) $x^2y'' - xy' + y = 0$

d) $x^2y'' + 3xy' + y = 0$

**Problema 16.** Resolver:

a) $x^2y'' + xy' + 4y = 0$ (raíces complejas)

b) $x^2y'' - 3xy' + 4y = 0$ (raíz doble)

**Problema 17.** Resolver el PVI: $x^2y'' - 2y = 0$, $y(1) = 1$, $y'(1) = 4$

## 2.5 Problemas de Valor Inicial

**Problema 18.** Resolver los PVI:

a) $y'' + 4y' + 3y = 0$, $y(0) = 0$, $y'(0) = 2$

b) $y'' - 6y' + 9y = 0$, $y(0) = 1$, $y'(0) = 4$

c) $y'' + 2y' + 2y = 0$, $y(0) = 1$, $y'(0) = -1$

**Problema 19.** Resolver los PVI:

a) $y'' + y = 1$, $y(0) = 0$, $y'(0) = 0$

b) $y'' - y = e^{-x}$, $y(0) = 1$, $y'(0) = 0$

c) $y'' + 4y = \cos 2x$, $y(0) = 0$, $y'(0) = 1$

## 2.6 Aplicaciones

### Oscilador Armónico

**Problema 20.** Una masa de 2 kg está unida a un resorte con $k = 18$ N/m. El sistema no tiene amortiguamiento.

a) Escribir la ecuación diferencial del movimiento

b) Si $x(0) = 0.1$ m y $\dot{x}(0) = 0$, encontrar $x(t)$

c) Determinar la frecuencia y período de oscilación

**Problema 21.** Un sistema masa-resorte tiene $m = 1$ kg, $c = 2$ N·s/m, $k = 5$ N/m.

a) Determinar el tipo de amortiguamiento

b) Resolver el PVI con $x(0) = 1$, $\dot{x}(0) = 0$

c) Encontrar la frecuencia de oscilación amortiguada

**Problema 22.** Un sistema críticamente amortiguado tiene $m = 1$ kg y $k = 4$ N/m.

a) Encontrar el valor de $c$ para amortiguamiento crítico

b) Resolver con $x(0) = 2$, $\dot{x}(0) = 0$

### Oscilaciones Forzadas

**Problema 23.** Un oscilador con $m = 1$ kg, $c = 0$, $k = 9$ N/m está sujeto a $F(t) = 10\cos 2t$ N.

a) Escribir y resolver la ecuación del movimiento

b) Si $x(0) = 0$ y $\dot{x}(0) = 0$, encontrar la solución particular

**Problema 24.** Un sistema con $m = 1$, $c = 2$, $k = 5$ está sujeto a $F = 10\cos t$.

a) Encontrar la solución de estado estacionario

b) Calcular la amplitud de oscilación forzada

### Circuitos RLC

**Problema 25.** Un circuito RLC tiene $L = 1$ H, $R = 6$ Ω, $C = 1/9$ F.

a) Escribir la ecuación para la carga $q(t)$

b) Determinar el tipo de respuesta

c) Resolver con $q(0) = 1$ C, $i(0) = 0$

**Problema 26.** Un circuito tiene $L = 0.5$ H, $R = 4$ Ω, $C = 0.05$ F, y $E(t) = 10$ V.

a) Encontrar la carga en estado estacionario

b) Resolver el PVI con $q(0) = 0$, $i(0) = 0$

**Problema 27.** Circuito con $L = 1$ H, $R = 0$, $C = 0.04$ F, y $E(t) = 100\cos 5t$ V.

a) ¿Hay resonancia? Explicar

b) Encontrar la solución para la corriente

## 2.7 Problemas Mixtos

**Problema 28.** Clasificar y resolver:

a) $y'' + 2y' + y = xe^{-x}$

b) $4x^2y'' + 8xy' + y = 0$

c) $y'' - 4y' + 4y = e^{2x}/x$ (usar variación de parámetros)

**Problema 29.** Encontrar una ecuación diferencial de segundo orden con coeficientes constantes cuya solución general sea:

a) $y = C_1 e^{3x} + C_2 e^{-x}$

b) $y = e^{2x}(C_1 \cos 3x + C_2 \sin 3x)$

c) $y = (C_1 + C_2 x)e^{-2x}$

**Problema 30.** Un resorte se estira 0.5 m por una masa de 2 kg ($g = 10$ m/s²). El sistema tiene amortiguamiento viscoso con $c = 8$ N·s/m.

a) Encontrar la constante del resorte $k$

b) Clasificar el amortiguamiento

c) Si se desplaza 0.1 m y se suelta desde el reposo, encontrar $x(t)$
