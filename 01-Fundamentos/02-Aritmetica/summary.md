# Resumen rápido — Aritmética

- **Orden de operaciones:** Paréntesis → Potencias/raíces → Multiplicación/División → Suma/Resta.
- **Propiedades clave:** conmutativa (suma, producto), asociativa (suma, producto), distributiva $a(b+c)=ab+ac$.
- **Fracciones:** $\dfrac{a}{b}+\dfrac{c}{d}=\dfrac{ad+bc}{bd}$; multiplicar numeradores y denominadores; para dividir, multiplicar por el recíproco.
- **Porcentajes:** $p\%$ de $x$ es $\dfrac{p}{100}x$; cambio porcentual $= \dfrac{\text{nuevo}-\text{original}}{\text{original}}\cdot100\%$.
- **Potencias y raíces:** $a^m a^n = a^{m+n}$; $(a^m)^n = a^{mn}$; $a^{-n} = \dfrac{1}{a^n}$; $\sqrt[n]{a^n}=|a|$ si $n$ es par.
