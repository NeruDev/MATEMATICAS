# Resumen rápido — Funciones

- **Definición:** Una función $f: A \to B$ asigna a cada elemento $x \in A$ exactamente un elemento $y \in B$.

$$f(x) = y$$

- **Dominio:** Conjunto de todos los valores de entrada válidos para la función.
- **Rango:** Conjunto de todas las salidas posibles de la función.

## Métodos clave:
1. **Prueba de la línea vertical:** Una curva es función si toda línea vertical la corta en máximo un punto.
2. **Encontrar dominio:** Identificar restricciones (denominadores ≠ 0, raíces ≥ 0, etc.).
3. **Evaluar funciones:** Sustituir el valor de entrada en la expresión.

## Fórmulas comunes:

| Tipo de función | Ejemplo | Dominio |
|-----------------|---------|---------|
| Lineal | $f(x) = mx + b$ | $\mathbb{R}$ |
| Cuadrática | $f(x) = ax^2 + bx + c$ | $\mathbb{R}$ |
| Racional | $f(x) = \frac{P(x)}{Q(x)}$ | $\{x : Q(x) \neq 0\}$ |
| Radical | $f(x) = \sqrt{x}$ | $[0, \infty)$ |

## Regla rápida:
- Si hay denominador: excluir valores que lo hagan cero.
- Si hay raíz cuadrada: el radicando debe ser ≥ 0.
