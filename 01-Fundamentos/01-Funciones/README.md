# Funciones

## Propósito del tema

<!--
HUMANO: ¿Por qué existe este tema y qué problemas permite resolver?
IA: Usa esto para justificar ejercicios y conexiones.
-->

Las funciones son el concepto fundamental del cálculo. Permiten:
- Modelar relaciones entre variables
- Describir fenómenos físicos, económicos y naturales
- Establecer las bases para límites, derivadas e integrales

Sin entender funciones, no es posible avanzar en el estudio del cálculo.

1. **Teoría:**
   - Definición formal de función
   - Dominio y rango
   - Tipos de funciones

2. **Métodos:**

   - Evaluar funciones
   - Componer funciones

3. **Problemas:**
   - Ejercicios de identificación de dominio y rango
   - Ejercicios de evaluación
   - Ejercicios de composición
