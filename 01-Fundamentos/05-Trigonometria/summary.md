# Resumen rápido — Trigonometría

- **Razones en triángulo rectángulo:** $\sin\theta = \dfrac{\text{op}}{\text{hip}}$, $\cos\theta = \dfrac{\text{ady}}{\text{hip}}$, $\tan\theta = \dfrac{\text{op}}{\text{ady}}$.
- **Identidad fundamental:** $\sin^2\theta + \cos^2\theta = 1$; $1 + \tan^2\theta = \sec^2\theta$.
- **Ángulos notables (grados/radianes):** $30^\circ=\tfrac{\pi}{6}$, $45^\circ=\tfrac{\pi}{4}$, $60^\circ=\tfrac{\pi}{3}$.
- **Conversión:** rad = grados $\times \tfrac{\pi}{180}$; grados = rad $\times \tfrac{180}{\pi}$.
- **Período:** seno/coseno periodo $2\pi$; tangente periodo $\pi$.
