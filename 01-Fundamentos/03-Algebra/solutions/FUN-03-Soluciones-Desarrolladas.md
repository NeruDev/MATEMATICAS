## 3.1 Expresiones algebraicas

**1)** *Contexto: Clasificamos por el n√∫mero de t√©rminos.*
- $5x^2y$: **Monomio** (1 t√©rmino)
- $3a - 2b + c$: **Trinomio** (3 t√©rminos)
- $x^2 - 4$: **Binomio** (2 t√©rminos)

**2)** *Contexto: T√©rminos semejantes tienen las mismas variables con los mismos exponentes.*
- $3x^2$ y $5x^2$ son semejantes
- $-2xy$ y $7xy$ son semejantes
- $x^2y$ no tiene semejante en el grupo

**3)** *Contexto: Sustituimos y evaluamos.*
$$2(-2)^2 - 3(-2)(3) + (3)^2 = 2(4) + 18 + 9 = 8 + 18 + 9 = \boxed{35}$$

**4)** *Contexto: El grado del [polinomio](../../..](../../../glossary.md)#polinomio) es el mayor grado de sus t√©rminos.*
- $5x^4y^2$: grado $4+2=6$
- $-3x^3y^3$: grado $3+3=6$
- $2x^2y$: grado $2+1=3$
- $-7$: grado $0$

**Grado del [polinomio](../../..](../../../glossary.md)#polinomio): 6**

**5)** *Contexto: Combinamos t√©rminos semejantes.*
$$4x^2 - 2x^2 + (-3x + 5x) + (7 - 3) = \boxed{2x^2 + 2x + 4}$$

**6)** *Contexto: Traducimos del espa√±ol al lenguaje algebraico.*

Sea $x$ el primer n√∫mero y $y$ el segundo: $\boxed{3x + y^2}$

**7)** *Contexto: Sustituimos $x$ por $(a+1)$.*
$$P(a+1) = (a+1)^2 - 3(a+1) + 2$$
$$= a^2 + 2a + 1 - 3a - 3 + 2 = \boxed{a^2 - a}$$

**8)** *Contexto: Ordenamos por potencias decrecientes de $x$.*
$$\boxed{-2x^3 - x^2 + 5x + 3}$$

**9)** *Contexto: Evaluamos y despejamos $k$.*
$$2(4) - k(2) + 3 = 7$$
$$8 - 2k + 3 = 7$$
$$2k = 4 \Rightarrow k = \boxed{2}$$

**10)** *Contexto: Usamos el binomio de Newton.*
$$(x + 2)^4 = x^4 + 4x^3(2) + 6x^2(4) + 4x(8) + 16$$
$$= x^4 + 8x^3 + 24x^2 + 32x + 16$$

Coeficiente de $x^3$: $\boxed{8}$

## 3.2 Operaciones con polinomios

**17)** *Contexto: Por el teorema del residuo, el residuo es $P(-1)$.*

**18)** *Contexto: Como $P(1) = 0$, $(x-1)$ es factor.*

## 3.5 Fracciones algebraicas

**43)** *Contexto: Factorizamos y simplificamos.*
$$\frac{x^2 - 9}{x^2 - 6x + 9} = \frac{(x+3)(x-3)}{(x-3)^2} = \boxed{\frac{x+3}{x-3}}, \quad x \neq 3$$

**44)** *Contexto: Factorizamos numerador y denominador.*
$$\frac{2(x^2 - 4)}{(x+2)^2} = \frac{2(x+2)(x-2)}{(x+2)^2} = \boxed{\frac{2(x-2)}{x+2}}, \quad x \neq -2$$

**45)** *Contexto: [MCM](../../..](../../../glossary.md)#mcm) de denominadores.*
$$\frac{2(x-1) + 3(x+1)}{(x+1)(x-1)} = \frac{2x - 2 + 3x + 3}{x^2 - 1} = \boxed{\frac{5x + 1}{x^2 - 1}}$$

**46)** *Contexto: [MCM](../../..](../../../glossary.md)#mcm) es $(x+2)(x-2)$.*
$$\frac{x(x-2) - 2(x+2)}{(x+2)(x-2)} = \frac{x^2 - 2x - 2x - 4}{x^2 - 4} = \boxed{\frac{x^2 - 4x - 4}{x^2 - 4}}$$

**47)** *Contexto: Factorizamos y multiplicamos.*
$$\frac{(x+2)(x-2)}{x+3} \cdot \frac{(x+3)(x-3)}{x-2} = \boxed{(x+2)(x-3)}$$

**48)** *Contexto: Dividir = multiplicar por rec√≠proco.*
$$\frac{(x+1)(x-1)}{x(x+1)} \cdot \frac{x}{x-1} = \boxed{1}, \quad x \neq 0, \pm 1$$

**49)** *Contexto: Multiplicamos por $xy$.*
$$\frac{\frac{y-x}{xy}}{\frac{y+x}{xy}} = \boxed{\frac{y-x}{y+x}}$$

**50)** *Contexto: El denominador com√∫n es $x^2 - 9 = (x-3)(x+3)$.*
$$\frac{2(x+3) + 1(x-3)}{(x-3)(x+3)} = \frac{6}{x^2-9}$$
$$2x + 6 + x - 3 = 6$$
$$3x = 3 \Rightarrow x = \boxed{1}$$

**51)** *Contexto: Excluimos valores que hacen cero el denominador.*
$$x^2 - 5x + 6 = (x-2)(x-3) = 0 \Rightarrow x = 2 \text{ o } x = 3$$

**[Dominio](../../..](../../../glossary.md)#dominio):** $\boxed{\mathbb{R} \setminus \{2, 3\}}$

**52)** *Contexto: Factorizamos usando suma de cubos.*
$$\frac{(x-2)(x^2+2x+4)}{(x+2)(x-2)} = \boxed{\frac{x^2+2x+4}{x+2}}, \quad x \neq 2$$
