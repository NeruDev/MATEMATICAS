## Soluci√≥n 1.1: M√©todo de Euler

**Problema:** $y' = 2xy$, $y(0) = 1$, calcular $y(0.5)$ con $h = 0.1$

**Soluci√≥n:**

$y_{n+1} = y_n + h \cdot f(x_n, y_n) = y_n + 0.1 \cdot 2x_ny_n$

| $n$ | $x_n$ | $y_n$ | $f_n = 2x_ny_n$ | $y_{n+1} = y_n + 0.1f_n$ |
|-----|-------|-------|-----------------|--------------------------|
| 0 | 0 | 1 | 0 | 1 |
| 1 | 0.1 | 1 | 0.2 | 1.02 |
| 2 | 0.2 | 1.02 | 0.408 | 1.0608 |
| 3 | 0.3 | 1.0608 | 0.6365 | 1.1244 |
| 4 | 0.4 | 1.1244 | 0.8996 | 1.2144 |
| 5 | 0.5 | 1.2144 | ‚Äî | ‚Äî |

**Resultado:** $y(0.5) \approx 1.2144$

**Valor exacto:** $y = e^{x^2}$, entonces $y(0.5) = e^{0.25} = 1.2840$

**Error:** $|1.2144 - 1.2840| = 0.0696$ (5.4%)

## Soluci√≥n 2.1: M√©todo de Heun

**Problema:** $y' = x^2 + y$, $y(0) = 1$, calcular $y(0.4)$ con $h = 0.2$

**Soluci√≥n:**

**Paso 0 ‚Üí 1 ($x: 0 \to 0.2$):**

$k_1 = f(0, 1) = 0^2 + 1 = 1$

$\tilde{y}_1 = 1 + 0.2(1) = 1.2$

$k_2 = f(0.2, 1.2) = 0.04 + 1.2 = 1.24$

$y_1 = 1 + \frac{0.2}{2}(1 + 1.24) = 1 + 0.1(2.24) = 1.224$

**Paso 1 ‚Üí 2 ($x: 0.2 \to 0.4$):**

$k_1 = f(0.2, 1.224) = 0.04 + 1.224 = 1.264$

$\tilde{y}_2 = 1.224 + 0.2(1.264) = 1.4768$

$k_2 = f(0.4, 1.4768) = 0.16 + 1.4768 = 1.6368$

$y_2 = 1.224 + \frac{0.2}{2}(1.264 + 1.6368) = 1.224 + 0.1(2.9008) = 1.5141$

**Resultado:** $y(0.4) \approx 1.5141$

**Comparaci√≥n con Euler ($h = 0.2$):**

Euler: $y_1 = 1 + 0.2(1) = 1.2$, $y_2 = 1.2 + 0.2(1.24) = 1.448$

Error Euler: mayor que Heun.

## Soluci√≥n 3.1: Runge-Kutta 4

**Problema:** $y' = y - x^2 + 1$, $y(0) = 0.5$, calcular $y(1)$ con $h = 0.2$

**Soluci√≥n exacta:** $y = x^2 + 2x + 1 - 0.5e^x$

**Paso 0 ‚Üí 1 ($x: 0 \to 0.2$):**

$k_1 = f(0, 0.5) = 0.5 - 0 + 1 = 1.5$

$k_2 = f(0.1, 0.5 + 0.1 \times 1.5) = f(0.1, 0.65) = 0.65 - 0.01 + 1 = 1.64$

$k_3 = f(0.1, 0.5 + 0.1 \times 1.64) = f(0.1, 0.664) = 0.664 - 0.01 + 1 = 1.654$

$k_4 = f(0.2, 0.5 + 0.2 \times 1.654) = f(0.2, 0.8308) = 0.8308 - 0.04 + 1 = 1.7908$

$y_1 = 0.5 + \frac{0.2}{6}(1.5 + 2(1.64) + 2(1.654) + 1.7908)$
$= 0.5 + \frac{0.2}{6}(9.8788) = 0.5 + 0.3293 = 0.8293$

| $n$ | $x_n$ | $y_n$ (RK4) | $y$ exacta |
|-----|-------|-------------|------------|
| 0 | 0 | 0.5 | 0.5 |
| 1 | 0.2 | 0.8293 | 0.8293 |
| 2 | 0.4 | 1.2141 | 1.2141 |
| 3 | 0.6 | 1.6489 | 1.6489 |
| 4 | 0.8 | 2.1272 | 2.1272 |
| 5 | 1.0 | 2.6408 | 2.6409 |

**Error en $y(1)$:** $|2.6408 - 2.6409| = 0.0001$

## Soluci√≥n 5.1: Sistema de EDO

**Problema:** $y'' + y = 0$, $y(0) = 1$, $y'(0) = 0$

**Conversi√≥n a sistema:**

Sea $z_1 = y$, $z_2 = y'$

$$\begin{cases} z_1' = z_2 \\ z_2' = -z_1 \end{cases}$$

Condiciones: $z_1(0) = 1$, $z_2(0) = 0$

**Euler con $h = 0.1$:**

$z_{1,n+1} = z_{1,n} + h \cdot z_{2,n}$

$z_{2,n+1} = z_{2,n} + h \cdot (-z_{1,n})$

| $n$ | $x_n$ | $z_{1,n} = y_n$ | $z_{2,n} = y'_n$ | $\cos(x_n)$ |
|-----|-------|-----------------|------------------|-------------|
| 0 | 0 | 1 | 0 | 1 |
| 1 | 0.1 | 1 | -0.1 | 0.9950 |
| 2 | 0.2 | 0.99 | -0.2 | 0.9801 |
| 3 | 0.3 | 0.97 | -0.299 | 0.9553 |

**$y(0.3) \approx 0.97$**, exacto: $\cos(0.3) = 0.9553$

Error: 0.0147 (1.5%)

## Soluci√≥n 7.1: Enfriamiento de Newton

**Problema:** $T' = -0.1(T - 20)$, $T(0) = 90$, encontrar cu√°ndo $T = 50¬∞C$

**RK4 con $h = 1$ min:**

$f(t, T) = -0.1(T - 20)$

**Paso 0 ‚Üí 1:**

$k_1 = -0.1(90 - 20) = -7$

$k_2 = -0.1(90 + 0.5(-7) - 20) = -0.1(66.5) = -6.65$

$k_3 = -0.1(90 + 0.5(-6.65) - 20) = -0.1(66.675) = -6.6675$

$k_4 = -0.1(90 + (-6.6675) - 20) = -0.1(63.3325) = -6.3333$

$T_1 = 90 + \frac{1}{6}(-7 + 2(-6.65) + 2(-6.6675) + (-6.3333))$
$= 90 + \frac{1}{6}(-39.9683) = 90 - 6.6614 = 83.3386$

| $t$ (min) | $T$ (¬∞C) |
|-----------|----------|
| 0 | 90 |
| 1 | 83.34 |
| 2 | 77.33 |
| 3 | 71.89 |
| ... | ... |
| 7 | 52.49 |
| 8 | 49.56 |

**Interpolando:** $T = 50¬∞C$ aproximadamente a $t \approx 7.9$ minutos.

**Verificaci√≥n anal√≠tica:** $T = 20 + 70e^{-0.1t}$

$50 = 20 + 70e^{-0.1t}$ ‚Üí $e^{-0.1t} = \frac{30}{70}$ ‚Üí $t = 10\ln(7/3) = 8.47$ min

## Soluci√≥n 6.1: An√°lisis de Estabilidad

**Problema:** $y' = -5y$, $y(0) = 1$

**a) Tama√±o de paso m√°ximo (Euler):**

Condici√≥n: $|1 + h\lambda| < 1$ con $\lambda = -5$

$|1 - 5h| < 1$

$-1 < 1 - 5h < 1$

$0 < 5h < 2$

$0 < h < 0.4$

**Paso m√°ximo:** $h_{max} = 0.4$

**b) Verificaci√≥n experimental:**

**$h = 0.3$ (estable):**

| $n$ | $y_n$ | $y$ exacta |
|-----|-------|------------|
| 0 | 1 | 1 |
| 1 | 0.4 | 0.223 |
| 2 | -0.02 | 0.050 |
| 3 | -0.128 | 0.011 |

Oscila pero converge.

**$h = 0.5$ (inestable):**

$y_1 = 1 + 0.5(-5)(1) = -1.5$

$y_2 = -1.5 + 0.5(-5)(-1.5) = 2.25$

$y_3 = 2.25 + 0.5(-5)(2.25) = -3.375$

**Diverge oscilando** ‚Üí Confirma inestabilidad para $h > 0.4$

## Soluci√≥n 3.4: Circuito RL

**Problema:** $L\frac{di}{dt} + Ri = V$ con $L = 2$, $R = 4$, $V = 12$, $i(0) = 0$

Reescribiendo: $\frac{di}{dt} = \frac{V - Ri}{L} = \frac{12 - 4i}{2} = 6 - 2i$

**RK4 con $h = 0.1$:**

$f(t, i) = 6 - 2i$

**Paso 0 ‚Üí 1:**

$k_1 = 6 - 2(0) = 6$

$k_2 = 6 - 2(0 + 0.05 \times 6) = 6 - 0.6 = 5.4$

$k_3 = 6 - 2(0 + 0.05 \times 5.4) = 6 - 0.54 = 5.46$

$k_4 = 6 - 2(0 + 0.1 \times 5.46) = 6 - 1.092 = 4.908$

$i_1 = 0 + \frac{0.1}{6}(6 + 2(5.4) + 2(5.46) + 4.908) = \frac{0.1}{6}(32.628) = 0.5438$

| $t$ | $i$ (A) | $i$ exacta |
|-----|---------|------------|
| 0 | 0 | 0 |
| 0.1 | 0.5438 | 0.5438 |
| 0.2 | 0.9675 | 0.9675 |
| ... | ... | ... |
| 1.0 | 2.7293 | 2.7293 |

**Soluci√≥n exacta:** $i = 3(1 - e^{-2t})$, $i(1) = 3(1 - e^{-2}) = 2.7293$ ‚úì
