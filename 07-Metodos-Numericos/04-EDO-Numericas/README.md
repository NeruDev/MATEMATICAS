# EDO Numéricas

## Descripción
Métodos numéricos para resolver ecuaciones diferenciales ordinarias con problemas de valor inicial. Incluye métodos de un paso (Euler, Heun, Runge-Kutta) y métodos multipaso (Adams-Bashforth, Adams-Moulton).

## Subtemas
1. Método de Euler (explícito e implícito)
2. Método de Heun (Euler mejorado)
3. Runge-Kutta de orden 2 (RK2)
4. Runge-Kutta de orden 4 (RK4)
5. Métodos multipaso: Adams-Bashforth
6. Métodos multipaso: Adams-Moulton
7. Sistemas de EDO
8. Análisis de estabilidad y error

## Prerrequisitos
- Ecuaciones diferenciales ordinarias
- Integración numérica
- Series de Taylor

## Aplicaciones
- Modelado de poblaciones
- Circuitos eléctricos
- Mecánica (movimiento de proyectiles, osciladores)
- Cinética química
- Transferencia de calor
