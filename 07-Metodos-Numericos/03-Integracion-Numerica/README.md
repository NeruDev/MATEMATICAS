# Integración Numérica

## Descripción

Métodos para aproximar integrales definidas $\int_a^b f(x)\,dx$ cuando no existe antiderivada elemental o la función se conoce solo en puntos discretos.

## Subtemas

1. **Regla del trapecio** - Aproximación lineal por tramos
2. **Regla de Simpson 1/3 y 3/8** - Aproximación cuadrática y cúbica
3. **Integración de Romberg** - Extrapolación de Richardson
4. **Cuadratura de Gauss-Legendre** - Nodos y pesos óptimos
5. **Integración compuesta** - Subdivisión del intervalo
6. **Análisis del error** - Cotas y convergencia
