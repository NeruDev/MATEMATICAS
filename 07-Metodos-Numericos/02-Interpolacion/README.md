# Interpolación

## Descripción

Métodos para construir funciones que pasan exactamente por un conjunto de puntos dados. Incluye polinomios de Lagrange, diferencias divididas de Newton y splines cúbicos.

## Subtemas

1. **Interpolación de Lagrange** - Construcción directa del polinomio interpolante
2. **Diferencias divididas de Newton** - Forma recursiva del polinomio
3. **Interpolación progresiva y regresiva** - Datos equiespaciados
4. **Error de interpolación** - Análisis del error y fenómeno de Runge
5. **Splines cúbicos** - Interpolación suave por tramos
6. **Interpolación de Hermite** - Incluye condiciones sobre derivadas
