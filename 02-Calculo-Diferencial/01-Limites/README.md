# Límites

## Propósito del tema

<!--
HUMANO: ¿Por qué existe este tema y qué problemas permite resolver?
IA: Usa esto para justificar ejercicios y conexiones.
-->

Los límites son el concepto fundamental del cálculo. Permiten:
- Estudiar el comportamiento de funciones cerca de un punto
- Definir continuidad de funciones
- Establecer las bases para derivadas e integrales
- Analizar comportamiento en el infinito

Los límites responden la pregunta: "¿A qué valor se aproxima $f(x)$ cuando $x$ se acerca a $a$?"

2. **Métodos:**

   - Factorización

   - L'Hôpital (referencia a derivadas)

3. **Problemas:**
   - Ejercicios de evaluación de límites
   - Ejercicios de formas indeterminadas
   - Ejercicios de límites en el infinito
