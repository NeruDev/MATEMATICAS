# Cheatsheet de Notación Matemática

## Conjuntos Numéricos

| Símbolo | Nombre | Descripción |
|---------|--------|-------------|
| $\mathbb{N}$ | Naturales | $\{1, 2, 3, ...\}$ |
| $\mathbb{Z}$ | Enteros | $\{..., -2, -1, 0, 1, 2, ...\}$ |
| $\mathbb{Q}$ | Racionales | Fracciones $\frac{p}{q}$ |
| $\mathbb{R}$ | Reales | Todos los números reales |
| $\mathbb{C}$ | Complejos | $a + bi$ |

## Funciones

| Notación | Significado |
|----------|-------------|
| $f: A \to B$ | Función de A en B |
| $f(x)$ | Valor de f en x |
| $f \circ g$ | Composición de f y g |
| $f^{-1}$ | Función inversa |
| $\text{Dom}(f)$ | Dominio de f |
| $\text{Ran}(f)$ | Rango de f |

## Límites

| Notación | Significado |
|----------|-------------|
| $\lim_{x \to a} f(x)$ | Límite de f cuando x tiende a a |
| $\lim_{x \to a^+}$ | Límite por la derecha |
| $\lim_{x \to a^-}$ | Límite por la izquierda |
| $\lim_{x \to \infty}$ | Límite en el infinito |

## Derivadas

| Notación | Significado |
|----------|-------------|
| $f'(x)$ | Derivada de f |
| $\frac{df}{dx}$ | Derivada de f respecto a x |
| $\frac{d^n f}{dx^n}$ | Derivada n-ésima |
| $f''(x)$ | Segunda derivada |

## Integrales

| Notación | Significado |
|----------|-------------|
| $\int f(x)\,dx$ | Integral indefinida |
| $\int_a^b f(x)\,dx$ | Integral definida de a a b |
| $F(x) + C$ | Antiderivada + constante |

## Vectores

| Notación | Significado |
|----------|-------------|
| $\vec{v}$ | Vector v |
| $\|\vec{v}\|$ | Magnitud de v |
| $\vec{u} \cdot \vec{v}$ | Producto punto |
| $\vec{u} \times \vec{v}$ | Producto cruz |
| $\nabla f$ | Gradiente de f |

## Operadores Lógicos

| Símbolo | Significado |
|---------|-------------|
| $\forall$ | Para todo |
| $\exists$ | Existe |
| $\in$ | Pertenece a |
| $\subset$ | Subconjunto |
| $\Rightarrow$ | Implica |
| $\Leftrightarrow$ | Si y solo si |
