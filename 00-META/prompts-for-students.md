# Prompts listos para estudiantes (copiar/pegar)

Usa estos prompts con tu asistente de IA (Copilot, ChatGPT, etc.) para aprovechar mejor el contenido del repositorio.

## 1. Modo tutor con quiz

```
Eres mi tutor — usando el archivo `theory/definicion-limite.md` actúa así: hazme un quiz de 5 preguntas conceptuales; espera mis respuestas y no des las soluciones hasta que solicite 'revelar'.
```

## 2. Generador de ejercicios

```
Genera 3 ejercicios de práctica nivel intermedio basados en `01-Limites/problems/` y devuelve solo enunciados y tags en JSON.
```

## 3. Simplificador de conceptos

```
Convierte `theory/definicion-limite.md` a lenguaje natural apto para explicar a alguien sin matemáticas en 6 frases.
```

## 4. Verificador de soluciones

```
Revisa mi solución paso a paso del problema en `problems/limite-funcion-racional.md` y dime si hay errores. No me des la respuesta, solo indícame dónde me equivoqué.
```

## 5. Mapa de dependencias

```
Analiza el `manifest.json` y `README.md` de este tema y dime qué prerequisitos necesito dominar antes de estudiarlo.
```

## 6. Resumen personalizado

```
Lee `summary.md` y genera una versión más corta de máximo 5 puntos para usar en una hoja de referencia de examen.
```
