# Funciones vectoriales de una variable real

## Propósito del tema

Analizar curvas espaciales usando funciones vectoriales para estudiar límites, continuidad, derivadas, integrales y propiedades geométricas como curvatura.

## Ruta de aprendizaje

1. **Definición y representación:** $\mathbf{r}(t) = \langle x(t), y(t), z(t) \rangle$.
2. **Límites y continuidad:** comportamiento de $\mathbf{r}(t)$ y sus componentes.
3. **Derivadas:** velocidad y aceleración; interpretación geométrica.
4. **Integración:** desplazamiento y longitud de arco mediante integrales de $\mathbf{r}(t)$.
5. **Vectores intrínsecos:** tangente, normal y binormal; marco de Frenet.
6. **Curvatura:** definición y cálculo a partir de $\mathbf{r}'(t)$ y $\mathbf{r}''(t)$.

## Conexiones

- **Prerequisitos:** vectores en $\mathbb{R}^3$, derivadas e integrales en una variable.
- **Usos posteriores:** campos vectoriales, integrales de línea y teoremas de integración múltiple.
