# Resumen rápido — Funciones de varias variables

- **Límite/continuidad:** se prueban en todas las direcciones; contrajemplos comunes con caminos diferentes.
- **Derivada parcial:** $f_x=\frac{\partial f}{\partial x}$, $f_y=\frac{\partial f}{\partial y}$; similares para $z$.
- **Diferencial:** $df = f_x\,dx + f_y\,dy$ (en 2D); aproxima $\Delta f$ para incrementos pequeños.
- **Regla de la cadena:** si $z=f(x,y)$ y $x=x(t)$, $y=y(t)$ entonces $\frac{dz}{dt}=f_x\,x'(t)+f_y\,y'(t)$.
- **Gradiente:** $\nabla f = \langle f_x, f_y, f_z \rangle$ apunta a la mayor tasa de cambio.
- **Extremos locales (2D):** puntos críticos donde $\nabla f=0$; usa el Hessiano para clasificar.
