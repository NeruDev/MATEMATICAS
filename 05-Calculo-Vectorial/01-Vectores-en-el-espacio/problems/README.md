## Nivel básico

### Magnitud y vectores unitarios
1. Dado $\mathbf{v} = \langle 3, -4, 12 \rangle$, calcula su magnitud y vector unitario.

2. Encuentra la magnitud del vector que va del punto $A = (1, 2, 3)$ al punto $B = (4, 6, 3)$.

3. Si $\mathbf{u} = \langle 2, -1, 2 \rangle$, ¿es un vector unitario? Si no, normalízalo.

### Operaciones básicas
4. Dados $\mathbf{a} = \langle 1, -2, 3 \rangle$ y $\mathbf{b} = \langle 4, 0, -1 \rangle$, calcula:
   - a) $\mathbf{a} + \mathbf{b}$
   - b) $2\mathbf{a} - 3\mathbf{b}$
   - c) $\lVert \mathbf{a} - \mathbf{b} \rVert$

5. Encuentra un vector de magnitud 10 en la dirección de $\mathbf{v} = \langle 1, 2, 2 \rangle$.

---

## Nivel intermedio

### Producto escalar y ángulos
6. Dados $\mathbf{u} = \langle 1, 2, -1 \rangle$ y $\mathbf{v} = \langle 2, -1, 2 \rangle$, encuentra el ángulo entre ellos.

7. Determina si los vectores $\mathbf{a} = \langle 3, -1, 2 \rangle$ y $\mathbf{b} = \langle 1, 1, -1 \rangle$ son ortogonales, paralelos o ninguno.

8. Encuentra el valor de $k$ para que $\mathbf{u} = \langle 2, k, -1 \rangle$ y $\mathbf{v} = \langle 1, -3, 2 \rangle$ sean perpendiculares.

### Proyecciones
9. Proyecta $\mathbf{u} = \langle 6, 3, 2 \rangle$ sobre $\mathbf{v} = \langle 2, 1, 2 \rangle$.

10. Un vector fuerza $\mathbf{F} = \langle 4, 3, 0 \rangle$ N actúa sobre un objeto. ¿Cuál es la componente de la fuerza en la dirección de $\mathbf{d} = \langle 1, 0, 0 \rangle$?

### Producto vectorial
11. Calcula $\mathbf{u} \times \mathbf{v}$ para $\mathbf{u} = \langle 1, 2, -1 \rangle$ y $\mathbf{v} = \langle 2, -1, 2 \rangle$.

12. Encuentra el área del triángulo con vértices $A = (1, 0, 0)$, $B = (0, 2, 0)$, $C = (0, 0, 3)$.

13. Determina un vector unitario perpendicular a ambos: $\mathbf{a} = \langle 1, 1, 0 \rangle$ y $\mathbf{b} = \langle 0, 1, 1 \rangle$.

---

## Nivel avanzado

### Rectas en el espacio
14. Encuentra las ecuaciones paramétrica y simétrica de la recta que pasa por $(1, 0, 2)$ con dirección $\langle 2, 1, -1 \rangle$.

15. La recta $L_1$ pasa por $(1, 2, 3)$ con dirección $\langle 1, -1, 2 \rangle$. La recta $L_2$ pasa por $(3, 0, 1)$ con dirección $\langle 2, -2, 4 \rangle$. ¿Son paralelas, coincidentes, secantes o alabeadas?

16. Encuentra el punto donde la recta $\mathbf{r}(t) = \langle 1+2t, t, 2-t \rangle$ interseca el plano $2x - y + 2z = 5$.

### Planos
17. Encuentra la ecuación del plano que contiene los puntos $A = (1, 1, 1)$, $B = (2, 0, 1)$, $C = (1, 2, 0)$.

18. Calcula la distancia del punto $(3, -1, 4)$ al plano $x + 2y - 2z = 7$.

19. Encuentra el ángulo entre los planos $2x - y + z = 3$ y $x + y + 2z = 1$.

20. Halla la ecuación del plano que pasa por $(2, 1, -1)$ y es paralelo al plano $3x - 2y + z = 5$.

### Problemas de aplicación
21. **Volumen**: Calcula el volumen del paralelepípedo formado por $\mathbf{u} = \langle 1, 0, 2 \rangle$, $\mathbf{v} = \langle 0, 3, 1 \rangle$, $\mathbf{w} = \langle 2, 1, 0 \rangle$.

22. **Trabajo**: Una fuerza $\mathbf{F} = \langle 3, 4, -2 \rangle$ N mueve un objeto del punto $A = (1, 0, 2)$ al punto $B = (4, 2, 0)$. Calcula el trabajo realizado (en Joules).

23. **Torque**: Una fuerza $\mathbf{F} = \langle 0, 5, 0 \rangle$ N se aplica en el punto $P = (3, 0, 0)$ respecto al origen. Calcula el vector torque $\boldsymbol{\tau} = \mathbf{r} \times \mathbf{F}$.
