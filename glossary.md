# Glosario de Matemáticas

## Términos Generales

| Término | Definición |
|---------|------------|
| Función | Relación que asigna a cada elemento de un conjunto exactamente un elemento de otro conjunto |
| Límite | Valor al que se aproxima una función cuando la variable independiente se acerca a un punto |
| Derivada | Tasa de cambio instantánea de una función |
| Integral | Operación inversa de la derivada; área bajo la curva |
| Vector | Cantidad con magnitud y dirección |

## Notación Común

| Símbolo | Significado |
|---------|-------------|
| $f(x)$ | Función de x |
| $\lim_{x \to a}$ | Límite cuando x tiende a a |
| $\frac{dy}{dx}$ | Derivada de y respecto a x |
| $\int$ | Integral |
| $\vec{v}$ | Vector v |
